# Flask Web Application

## Overview
This project is a **Flask-based web application** built to demonstrate backend web development using Python.  
It focuses on creating a lightweight, structured web app with routing, templates, and dynamic content.

**Repository:** https://github.com/mariesitar/flask  
**Project Type:** Web Application / Backend Development

---

## Purpose
The purpose of this project is to:
- practice building web applications using Flask
- understand backend routing and request handling
- connect Python logic to HTML templates
- organize a project using best practices for Flask apps

This project highlights skills relevant to **web development, business analytics tools, and full-stack foundations**.

---

## Features
- Flask application structure
- Multiple routes/pages
- Dynamic rendering using Jinja templates
- Organized project files
- Ready for future expansion (databases, forms, APIs)

---

## Tech Stack
- **Python**
- **Flask**
- **HTML**
- **CSS**
